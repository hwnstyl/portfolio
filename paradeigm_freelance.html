Paradeigm Freelance Project

The scope of this project was to create a blog posting feature for Paradeigm LLC's customer, San Diego Brewers Guild. The blogging feature includes a blog post creation and editing platform for administration, and a public facing site for viewing the blog posts. The backend for the site, including: getting, posting, editing, and deleting endpoints, were created from scratch using NodeJS. The blog post creation and editing forms were created using Quill an open source WYSIWYG editor. The components were styled using Foundation as a grid framework.

Blog Post Creating & Editing

Pictured below is the administration view of the blog posts. Below the navigation bar, in the top left corner is a button that allows the user to create a new blog post. The blog post feed shows the blog post feature image, creation date, title, and tags. In the top right hand corner of each blog post in the feed there are three icons for editing, making the post visible/invisible on the public facing site, and deleting that post.

Also shown below is the blog post creation form and the form for editing a specific post. The form allows the user to set a title, slug (for creating the unique url), body, feature image, and tags for each post. Each of these inputs can be edited for any post.

Blog Post Viewing

These images show the public facing blog post viewing area. In the feed, a user can browse through blog posts from newest to oldest. Each post in the feed shows the creation date, title, tags, and feature image. When a user selects a specific post the post is shown and the next two most recent posts can be navigated to at the bottom of the post.
